OCW-CS-Track¶

Computer System Track¶

The sys track provides hands-on experience with foundation courses including four system components: operating systems, database management systems, networks, and distributed systems.

  1. 6.s081 20fall: roadmap
  2. 6.824: Paxos, Raft-kvs
  3. 15-445: B+Tree
  4. 15-441: STP, TCP, proxy, load balancer
  5. 6.826: disk replication correctness

Machine Learning Track¶

The goal of ml-track is to achieve solid understanding of machine learning systems through practices. It is tailed for people who may have studied for computer system track but lacked experience in machine learning.

  1. Nerual Networks by Andrej Karpathy
  2. tiny-llm by Alex Chi (sys view)
  3. CS336 (model view)
  4. 6.7900 (theory-heavy)
  5. dlsys (sys view)
